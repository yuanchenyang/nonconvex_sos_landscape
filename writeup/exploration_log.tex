\section*{Current Strategy}
The current proof search is split into two layers:
\begin{enumerate}
\item isolate the abstract SOCP certificate lemmas that depend only on the
definitions in \texttt{TernaryQuartic.lean};
\item use Julia to determine what algebraic decomposition should replace the
univariate factor-peeling argument in the ternary quartic rank-4 setting.
\end{enumerate}

The key modeling difference from the univariate reference is that the SOCP
conditions only quantify over admissible directions, i.e. rank-4 vectors of
quadratic polynomials. Any image-plus-cone certificate therefore has to keep all
witness directions quadratic.

\section*{Current Hypothesis}
The abstract SOCP step is not the blocker. The missing ingredient is a
ternary-quartic algebraic statement of the following form:
\[
  p \in \operatorname{im}(A_{u_{\mathrm{img}}}) +
      \operatorname{cone}(\sigma(\ker A_u))
\]
with all witnesses admissible, for every admissible rank-4 factor $u$ and every
SOS quartic $p$.

This is the multivariate replacement for the univariate factor-peeling/hgroup
reduction. The admissibility restriction rules out naively importing the
univariate kernel generators, because multiplying a quadratic factor by a
nonconstant polynomial leaves the quadratic direction space.

The current best candidate for the remaining proof is a normal-form
classification by the filtered dimensions
\[
  \dim(W \cap \mathbb{R}), \qquad \dim(W \cap \mathbb{R}[x_1,x_2]_{\le 1}),
  \qquad W = \operatorname{span}(u_1,\dots,u_4).
\]
The newly formalized dependent case handles $\dim W < 4$, and the surjective
case handles image rank $15$. The whole
$\dim(W \cap \mathbb{R}[x]_{\le 1}) = 3$ branch is now solved in Lean through
an abstract ``contains $1,x_1,x_2$'' theorem, not just at one representative.
The unresolved cases are therefore normalization statements reducing the
independent $\dim = 2$ configurations of image rank $13$ or $14$ to the
already-formalized representatives.

Current canonical candidates after orthogonal mixing of the four coordinates and
affine changes of variables:
\begin{itemize}
\item $\dim(W \cap \mathbb{R}[x]_{\le 1}) = 3$:
  \[
    u \sim (1, x_1, x_2, h), \qquad h \in \mathbb{R}[x_1,x_2]_2^{\mathrm{hom}}.
  \]
  More strongly, this branch no longer needs exact normalization to that
  representative. It suffices that $1,x_1,x_2 \in W$. The kernel family is:
  if $h_0 = x_1 a + x_2 b$ with $a,b$ linear, then
  \[
    w^{\mathrm{raw}}
    =
    \operatorname{rel}(-c_0,h_0)+\operatorname{rel}(c_1,a)+\operatorname{rel}(c_2,b),
  \]
  where $\sum_i c_{0,i}u_i = 1$, $\sum_i c_{1,i}u_i = x_1$, and
  $\sum_i c_{2,i}u_i = x_2$. After scaling by
  $\bigl(\sqrt{\sum_i c_{0,i}^2}\bigr)^{-1}$, the corrected square has degree
  at most $3$, so every quadratic square is in
  $\operatorname{im}(A_u)+\operatorname{cone}(\sigma(\ker A_u))$ without
  further normalization data.
\item $\dim(W \cap \mathbb{R}[x]_{\le 1}) = 2$, common-factor case:
  \[
    u \sim (x_1, x_2, x_1^2, x_1 x_2).
  \]
  The image misses exactly the classes of $1$ and $x_2^4$ in the monomial
  model. Explicit kernel directions:
  \[
    w_c = (-2x_2, x_1, 0, 1), \qquad
    w_q = (-x_2^2, x_1 x_2, 0, 0).
  \]
  Then $\sigma(w_c) \equiv 1$ and $\sigma(w_q) \equiv x_2^4$ modulo
  $\operatorname{im}(A_u)$.
\item $\dim(W \cap \mathbb{R}[x]_{\le 1}) = 2$, coprime case:
  \[
    u \sim (x_1, x_2, x_1^2, x_2^2)
  \]
  or any equivalent rank-$14$ model. The image already contains every positive
  degree monomial; only the constant class remains, and it is represented by
  \[
    w = (-x_1, 0, 1, 0), \qquad \sigma(w)=x_1^2+1 \equiv 1.
  \]
\item $\dim(W \cap \mathbb{R}[x]_{\le 1}) = 1$, constant case:
  \[
    u \sim (1, x_1^2, x_1 x_2, x_2^2),
  \]
  which is already surjective.
\item $\dim(W \cap \mathbb{R}[x]_{\le 1}) = 1$, linear case:
  \[
    u \sim (x_1, x_1^2, x_1 x_2, x_2^2).
  \]
  The image misses exactly the classes of $1$ and $x_2$. For any affine linear
  $\ell = c + d x_2$,
  \[
    w_\ell = (-x_2 \ell, 0, \ell, 0)
  \]
  lies in the kernel, and
  \[
    \sigma(w_\ell) = (1+x_2^2)\ell^2 \equiv c^2 + 2 c d\, x_2
    \pmod{\operatorname{im}(A_u)}.
  \]
  This matches the quotient classes of an arbitrary quadratic square.
\item \textbf{Correction.} The previous five-case list is not complete. When
  $\dim(W \cap \mathbb{R}[x]_{\le 1}) = 2$ and $1 \in W$, there is an
  additional mixed-affine subbranch. The representative probes
  \[
    (1,x_1,x_1^2,x_1x_2),\qquad
    (1,x_1,x_1x_2,x_2^2),\qquad
    (1,x_1,x_1^2,x_2^2)
  \]
  have image ranks $13$, $14$, and $15$ respectively, with missing monomial
  classes
  \[
    \{x_2^3,x_2^4\},\qquad \{x_1^4\},\qquad \varnothing.
  \]
  The first two also admit explicit kernel probes:
  \[
    w = (0,0,-x_2^2,x_1x_2)
  \]
  for the rank-$13$ model, giving
  \[
    A_u(w)=0,\qquad \sigma(w)=x_2^4+x_1^2x_2^2,
  \]
  and
  \[
    w = (0,0,-x_1x_2,x_1^2)
  \]
  for the rank-$14$ model, giving
  \[
    A_u(w)=0,\qquad \sigma(w)=x_1^2x_2^2+x_1^4.
  \]
  Modulo $\operatorname{im}(A_u)$ these produce the missing $x_2^4$ and $x_1^4$
  classes, and the rank-$13$ family extends to
  \[
    w_\ell=(0,0,-x_2\ell,x_1\ell),\qquad \ell=c+d x_2,
  \]
  which generates both missing classes $x_2^3$ and $x_2^4$ modulo the image.
  A further parabolic probe of the same branch gives
  \[
    (1,x_1,x_1x_2,x_1^2-x_2^2),\qquad
    (1,x_1,x_1x_2,x_1^2+x_2^2)
  \]
  both with image rank $15$. This suggests the following sharper
  classification: after fixing
  \[
    W \cap \mathbb{R}[x_1,x_2]_{\le 1} = \operatorname{span}(1,x_1),
  \]
  the remaining quadratic plane is the kernel of a binary quadratic
  $a x_1^2 + b x_1x_2 + c x_2^2$, and only the annihilators $x_2^2$ and
  $x_1^2$ give nonsurjective image ranks ($13$ and $14$ respectively). The
  new annihilator-family sweep in the checked-in Julia script supports the full
  completion-of-squares reduction:
  \begin{itemize}
  \item if $c \neq 0$, shear $x_2 \mapsto x_2 - \frac{b}{2c}x_1$ kills the
    cross term;
  \item if the resulting $x_1^2$ coefficient vanishes, the branch is the
    rank-$13$ model $x_2^2$;
  \item if that coefficient is nonzero, rescaling gives $x_1^2 \pm x_2^2$,
    numerically always surjective;
  \item if $c = 0$ but $b \neq 0$, the model reduces to $x_1x_2$, again
    numerically surjective;
  \item if $c = 0$ and $b = 0$, the model is the rank-$14$ case $x_1^2$.
  \end{itemize}
  The shear-and-scale formulas underlying this reduction are now formalized in
  \texttt{TernaryQuarticProof/MixedAffineNormalization.lean}. That file proves
  the explicit affine identities sending
  \[
    a x_1^2 + b x_1x_2 + c x_2^2
  \]
  to
  \[
    \Bigl(a - \frac{b^2}{4c}\Bigr) x_1^2 + c x_2^2
    \qquad (c \neq 0),
  \]
  and to
  \[
    b x_1x_2
    \qquad (c=0,\ b \neq 0),
  \]
  inside the affine subgroup preserving the line $\mathbb{R}x_1$.
  In this round the same file is tightened further: it now also proves the
  diagonal sign normalizations
  \[
    a x_1^2 + c x_2^2 \leadsto a(x_1^2+x_2^2)
    \qquad \text{when } a/c>0,
  \]
  and
  \[
    a x_1^2 + c x_2^2 \leadsto a(x_1^2-x_2^2)
    \qquad \text{when } -a/c>0,
  \]
  via the scalings
  \(x_2 \mapsto \sqrt{a/c}\,x_2\) and
  \(x_2 \mapsto \sqrt{-a/c}\,x_2\), and it combines those with the shear step
  into explicit shear-plus-scale normal-form equalities. So the algebraic
  normalization layer now already covers the three post-shear outcomes that the
  mixed-affine classification needs: rank-$14$, sum-square, and split-diagonal.
  So the remaining classification gap is specifically the constant-containing
  $\dim = 2$ branch rather than the whole independent nonsurjective case.
\end{itemize}

The remaining gap is no longer the representative algebra in these cases; it is
the normalization step that extracts, from a general admissible factor, the
quadratic annihilator and the coefficient vectors needed to apply those
formalized shear/scale identities inside Lean. After that data is produced, the
transport layer can pull back the already-formalized representative theorems.

One smaller alternative is now on the table for the mixed-affine branch:
replace exact representative transport by basis-invariant plane theorems.
For a plane of the form
\[
  x_1 \cdot \operatorname{span}(x_1,x_2)
  \qquad\text{or}\qquad
  x_2 \cdot \operatorname{span}(x_1,x_2),
\]
the image part depends only on the plane itself, not on the chosen basis of
its two coordinates. If one chooses an orthonormal coefficient-space basis of
that plane, the same syzygy as in the representative proofs produces kernel
families with a clean
\[
  \sigma = (a^2+b^2)\ell^2
\]
formula. This may avoid the last orthogonal-normalization obstruction in the
constant-containing $\dim = 2$ branch.

The first formal Lean infrastructure for that alternative is now added in
\texttt{TernaryQuarticProof/RepresentativeMixedAffinePlane.lean}. It proves:
\begin{itemize}
\item any explicit scalar relation $\sum_i c_i u_i=r$ produces an admissible
  image witness for $r q$;
\item if coefficient vectors $c,d \in \mathbb{R}^4$ are orthonormal, then
  their relation directions satisfy
  \[
    \sigma\bigl(\mathrm{rel}_c(p)+\mathrm{rel}_d(q)\bigr)=p^2+q^2.
  \]
\item the whole image half of the rank-$14$ mixed-affine representative proof
  is now abstracted away from the concrete representative: if a factor admits
  explicit relations for $1$, $x_1$, $x_1x_2$, and $x_2^2$, then every quartic
  with zero $x_1^4$ coefficient lies in its admissible image.
\item the residual argument for that branch is now abstracted as well: with the
  same relation data and an orthonormal coefficient-space pair for the
  $(x_1x_2,x_2^2)$ directions, the kernel
  \[
    w=\mathrm{rel}_{-c_2}(t x_1x_2)+\mathrm{rel}_{c_3}(t x_1^2)
  \]
  has
  \[
    \operatorname{coeff}_{x_1^4}(\sigma(w))=t^2
  \]
  and the admissible image-plus-cone certificate gives zero residual.
\item the rank-$13$ branch is now abstracted in the same basis-invariant way:
  from explicit relations for $1$, $x_1^2$, and $x_1x_2$, together with an
  orthonormal coefficient-space pair for the last two directions, the file now
  proves both the codimension-two image theorem and the full SOCP certificate.
  The abstract kernel is
  \[
    w=\mathrm{rel}_{-c_2}(x_2\ell)+\mathrm{rel}_{c_3}(x_1\ell),
    \qquad \ell=b+c x_2,
  \]
  and its $(x_2^3,x_2^4)$ coefficients match the representative formulas
  $(2bc,c^2)$.
\item the same file now also formalizes the surjective rank-$15$ plane orbits
  directly: any invertible basis of
  \[
    \operatorname{span}(x_1^2,x_2^2)
    \qquad\text{or}\qquad
    \operatorname{span}(x_1x_2,\ x_1^2-x_2^2)
  \]
  yields a no-spurious-SOCP theorem, and both statements are wrapped through
  the affine-equivalence transport layer.
\item the stronger follow-up abstraction is now split in two. The fully
  arbitrary orthonormal-basis version for the canonical planes
  $x_1 \cdot \operatorname{span}(x_1,x_2)$ and
  $x_2 \cdot \operatorname{span}(x_1,x_2)$ still ran into unstable coefficient
  bookkeeping for products of general linear forms and remains reverted. But a
  cleaner partial theorem is now formalized: if the quadratic relations differ
  from the monomial pairs $(x_1x_2,x_2^2)$ or $(x_1^2,x_1x_2)$ by an orthogonal
  $2 \times 2$ basis change inside that plane, then Lean reconstructs the
  exact monomial relations and reduces back to the existing rank-$14$ or
  rank-$13$ mixed-affine theorem. So the obstruction has narrowed from
  ``arbitrary plane-basis algebra'' to ``produce orthogonal in-plane basis
  data from a general mixed-affine factor.''
\item that new partial strengthening is now connected to the affine transport
  layer as well: the same file now wraps the rank-$14$ and rank-$13$ orthogonal
  in-plane basis theorems through an arbitrary polynomial algebra equivalence.
  In practical terms, once an affine normalization sends a mixed-affine factor
  to one of these orthogonal plane-basis configurations, Lean can conclude
  zero residual for the original SOCP point without building a separate
  transport proof each time.
\end{itemize}
So the remaining work for the plane strategy is now geometric rather than
algebraic: identify the right orthonormal relation data canonically from a
general mixed-affine factor.

The newest strategy adjustment is therefore to split the old ``normalization''
problem into two formally separate layers:
\begin{itemize}
\item affine variable-change transport on the polynomial side;
\item orthogonal factor-coordinate mixing on the rank-$4$ side.
\end{itemize}
The second layer was already formalized in
\texttt{TernaryQuarticProof/FactorTransform.lean}. The first layer is now
itself split into a homomorphism file
\texttt{TernaryQuarticProof/AffineTransform.lean} and an equivalence-transport
file \texttt{TernaryQuarticProof/AffineSocpTransform.lean}. This shift came
from a concrete obstruction: orthogonal mixing acts on coefficient vectors in
$\mathbb{R}^4$, so it does not by itself force exact coordinates $1$, $x_1$,
or $x_2$ to appear even when the corresponding affine lines lie in
$\operatorname{span}(u_1,\dots,u_4)$.

The next proof-serving abstraction is therefore more concrete: introduce an
explicit six-coordinate normal form for quadratics,
\[
  a_{00} + a_{10} x_1 + a_{01} x_2 + a_{20} x_1^2 + a_{11} x_1x_2 + a_{02} x_2^2,
\]
and formalize how the affine subgroup preserving $\mathbb{R}x_1$ acts on those
coordinates. The aim is not a generic symbolic algebra package; it is a small
normalization API tailored to the remaining constant-containing mixed-affine
branch. In particular, one wants explicit formulas for:
\begin{itemize}
\item translation in $x_2$, to absorb affine tails into the already-controlled
  $\operatorname{span}(1,x_1)$ part;
\item shear $x_2 \mapsto x_2 + t x_1$, to perform the completion-of-squares
  reduction on the homogeneous coefficients;
\item scaling $x_2 \mapsto s x_2$, to normalize the remaining nonzero diagonal
  coefficient.
\end{itemize}
This should reduce the last unresolved branch to coefficient extraction and a
binary-quadratic orbit calculation, rather than to another ad hoc representative
proof.

That coefficient layer is now strengthened in Lean. The same file
\texttt{TernaryQuarticProof/QuadraticCoordinateForm.lean} now also:
\begin{itemize}
\item defines the actual affine equivalence for translation $x_2 \mapsto x_2+t$
  inside the existing transport framework;
\item proves generic coefficient identities for arbitrary quadratics under that
  translation, for example
  \[
    \operatorname{coeff}_{x_2}(q(x_1,x_2+t))
    =
    \operatorname{coeff}_{x_2}(q)+2t\,\operatorname{coeff}_{x_2^2}(q),
  \]
  while the homogeneous coefficients
  \[
    \operatorname{coeff}_{x_1^2},\ \operatorname{coeff}_{x_1x_2},\ \operatorname{coeff}_{x_2^2}
  \]
  remain unchanged;
\item proves the analogous generic coefficient-update rules under shear and
  scaling.
\end{itemize}
So the mixed-affine normalization problem is now expressed in Lean directly at
the level of coefficient functionals on arbitrary quadratics, rather than only
through hand-expanded representative formulas.

The next strategy refinement is now very narrow: formalize the coefficient-
selected normalization parameters themselves. For a quadratic with coefficients
\[
  (a_{00},a_{10},a_{01},a_{20},a_{11},a_{02}),
\]
the planned Lean lemmas are exactly the scalar choices used in the paper proof:
\begin{itemize}
\item if $a_{02}\neq 0$, translate by
  \[
    t_{\mathrm{tr}}=-\frac{a_{01}}{2a_{02}}
  \]
  to kill the $x_2$ term;
\item if $a_{02}\neq 0$, shear by
  \[
    t_{\mathrm{sh}}=-\frac{a_{11}}{2a_{02}}
  \]
  to kill the $x_1x_2$ term and replace the $x_1^2$ coefficient by
  \[
    a_{20}-\frac{a_{11}^2}{4a_{02}};
  \]
\item if $a_{02}=0$ but $a_{11}\neq 0$, shear by
  \[
    t_{\mathrm{sh}}=-\frac{a_{20}}{a_{11}}
  \]
  to kill the $x_1^2$ term and reduce to the pure-cross orbit.
\end{itemize}
This algebra is no longer only a blueprint item: the same file now packages it
as the Lean theorem
\texttt{mixedAffineAnnihilator\_normal\_form\_cases}, giving a direct case split
on the annihilator coefficients.

\section*{New Lean Strategy Adjustment}
The first representative theorem suggests a better formalization pattern than
generic coefficient reconstruction. For explicit normal forms, it is often
easier to prove either:
\begin{itemize}
\item full image surjectivity by reducing quartics to monomial cases; or
\item an image-plus-kernel decomposition for each quadratic square
  $q^2$, using a decomposition $q = r + t$ where $r$ already lies in the span
  of the factor coordinates and $t$ is the small missing piece.
\end{itemize}

This avoids the failed earlier attempt to build a large generic monomial-basis
library first.

The next candidate abstraction was smaller and more targeted than the abandoned
generic coefficient library: prove a \emph{quadratic normal-form} lemma for
two-variable quadratics by using the support restriction coming from
\texttt{IsQuadratic} and the fact that every support monomial of total degree
at most $2$ is one of exactly six finitely many \texttt{Fin 2 ->0 Nat} patterns.

This step is now formalized in
\texttt{TernaryQuarticProof/QuadraticNormalForm.lean}. The file introduces the
six quadratic monomial indices
\[
  m_{00},\ m_{10},\ m_{01},\ m_{20},\ m_{11},\ m_{02},
\]
proves a case split for all exponent vectors of total degree at most $2$,
proves that the support of any quadratic polynomial is contained in this six
element set, and derives the six-monomial sum formula
\[
  q = \sum_{d \in \{m_{00},m_{10},m_{01},m_{20},m_{11},m_{02}\}}
      \operatorname{monomial}(d,\operatorname{coeff}_d(q)).
\]

The new proof strategy shift is therefore no longer ``build a quadratic normal
form'' but ``use the formalized quadratic normal form to drive a codimension-one
or codimension-two image argument for the mixed-affine representatives.'' The
first target
\[

\]
is now complete: the codimension-one image theorem, the coefficient-of-square
lemma for the missing $x_1^4$ class, and the explicit kernel correction have
all been formalized, giving a full representative-level no-spurious-SOCP
theorem.

That second mixed-affine target is now complete as well. So the constant-
containing mixed-affine branch is no longer blocked on representative algebra:
the two nonsurjective representatives

  \operatorname{span}(x_1x_2,\ x_1^2-x_2^2).
\]

The low-affine representative analysis is now complete as well in

sub-branch is now sharper than that wording suggests: there the common-
translation lemma and its integration are done, so the remaining work is to
extract exact normalized coordinates of the form \(u=(1,x_1,q_2,q_3)\), or
equivalently exact relation data for that same tuple, together with the same
determinant condition from the general branch. On the determinant-zero side,
the remaining work is now only extraction and transport through the coarse
split on the extracted line \(H(q_2,q_3)\): the cross-like
\(\operatorname{coeff}_{x_1^2} H=0\), \(\operatorname{coeff}_{x_0x_1} H\neq 0\)
branch and the complementary \(\operatorname{coeff}_{x_1^2} H\neq 0\) branch
both have internal shear normalization already built into Lean, while the
remaining pure-\(x_0^2\) branch now looks numerically like a generic rank-\(15\)
family plus a special low-affine rank-\(13\) sub-slice.
\item Extract from a general admissible factor the coefficient vectors and
annihilator data needed to feed that classification into the existing
transport lemmas; for the remaining mixed-affine and low-affine branches this
now only needs determinant data inside the canonical homogeneous plane or
triple type.
\item After that branch is settled, assemble the final theorem in
\texttt{TernaryQuarticProof.lean} and rerun the verification harness.
\end{itemize}
